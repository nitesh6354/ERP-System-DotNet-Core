\## üß± Architecture (Layered)



The solution is structured into the following layers:



\- \*\*ERP.API\*\*  

&nbsp; Presentation layer responsible for handling HTTP requests, authentication,

&nbsp; and responses.



\- \*\*ERP.Application\*\*  

&nbsp; Contains business logic, use cases, validations, and service interfaces.



\- \*\*ERP.Domain\*\*  

&nbsp; Core domain entities and business models.  

&nbsp; This layer has no dependency on any framework.



\- \*\*ERP.Infrastructure\*\*  

&nbsp; Handles database access using \*\*Entity Framework Core\*\* and

&nbsp; SQL Server Express.
